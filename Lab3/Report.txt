Выбран Homo sapiens
Инструмент картирования - minimap2
Фреймворк для пайплайнов - flyte 

Ссылка на риды (homo sapiens): https://trace.ncbi.nlm.nih.gov/Traces/?run=SRR32806055
Ссылка на референсный геном (та же, что в условии задачи):  https://hgdownload.soe.ucsc.edu/goldenPath/hg38/bigZips/hg38.fa.gz

bash-скрипты:
index.sh - индексирование рефернсного генома
eval.sh - скрипт разбора результатов samtools flagstat для получения % картированных ридов
pipeline.sh - скрипт, объединяющий все этапы оценки картирования. В нем в т.ч. проходит и аналог index.sh (при необходимости), и eval.sh для разбора результатов samtools flagstat

Для запуска пайплайна необходимо иметь установленными minimap2, samtools, freebayes
FastQC должен быть разархивирован в папку Lab3/FastQC (архив лежит здесь же)


Для установки фреймворка достаточно в папке flyte_code запустить setup.sh (требуется установленный python 3.11)

Для запуска пайплайнов:
source flyte-env311/bin/activate

flyte run --local hello.py hello_world_wf
ИЛИ
flyte run --local pipeline.py main


Пайплайн на фреймворке (pipeline.py) делает всё то же, что и pipeline.sh.
Результаты работы и графическое представление пайплайна лежат в Lab3/flyte

Элементы пайплайна намеренно делались как отражение алгоритма из блок-схемы, так что визуалиазция пайплайна и блок-схема алгоритма почти идентичны с точностью до перемены названий блоков. Главное отличие - что пайплайн дополнительно проводит индексирование при необходимости перед следованием по алгоритму блок=схемы.
